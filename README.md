=========================================
      STUDENT MARKS MANAGER (PYTHON)
=========================================

📌 Project Description
----------------------
Student Marks Manager is a beginner-friendly Python project that allows users to manage student records and marks using a simple menu-driven program.

The project demonstrates the use of Python fundamentals such as variables, lists, loops, conditional statements, and basic list operations.

-----------------------------------------
FEATURES
-----------------------------------------

1. Add a new student with marks.
2. View all student records.
3. Search for a student by name.
4. Calculate the average marks.
5. Display the student with the highest marks.
6. Exit the program safely.

-----------------------------------------
CONCEPTS USED
-----------------------------------------

- Python Variables
- Input and Output
- Data Types
- Lists
- List Methods (append, index)
- if, elif, else
- while Loop
- for Loop
- Built-in Functions
  - len()
  - sum()
  - max()

-----------------------------------------
REQUIREMENTS
-----------------------------------------

- Python 3.x
- Any Python IDE (VS Code, PyCharm, IDLE, etc.)

-----------------------------------------
HOW TO RUN
-----------------------------------------

1. Install Python 3.
2. Download or clone this repository.
3. Open the project in VS Code or any Python IDE.
4. Run the Python file.

Command:

python student_marks_manager.py

-----------------------------------------
MENU

1. Add Student
2. View Students
3. Search Student
4. Calculate Average
5. Highest Marks
6. Exit
